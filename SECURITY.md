# Security Policy

## Supported Versions

This is the time versions are supported:

| Type of Version | Time supported     |
| --------------- | ------------------ |
| LTS             | Ten years          |
| Release         | Five years         |
| Bugfix          | One month          |
| Nightly         | :x:                |

## Reporting a Vulnerability

Report vulnerabilities at [report-vulnerability@nainlang.xyz](mailto:report-vulnerability@nainlang.xyz)
